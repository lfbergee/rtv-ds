# Knapper

Vi har mange knapper

## PrimaryButton

Hovedknappen brukes for å markere den viktigste/ønskede operasjonen for brukeren

```tsx live=true
<PrimaryButton postfix="" isLoading={false} disabled={false}>
  Hello world
</PrimaryButton>
```

## SecondaryButton

Sekundærknappen brukes til andre operasjoner enn den viktigste

```tsx live=true
<SecondaryButton postfix="" isLoading={false} disabled={false}>
  Hello world
</SecondaryButton>
```

## TertiaryButton

Tertiærknappen brukes til destruktive eller mindre viktige handliger

```tsx live=true
<TertiaryButton postfix="" isLoading={false} disabled={false}>
  Hello world
</TertiaryButton>
```

## Button options

### With postfix

Kan brukes til å vise pris eller annen viktig info

```tsx render=true
<PrimaryButton postfix="249 kr">
  Film og Serier
</PrimaryButton>
```


### With icon

```tsx render=true
<PrimaryButton icon={Play}>
  Spill av
</PrimaryButton>
```

### With postfix and icon

```tsx render=true
<PrimaryButton icon={Play} postfix="249 kr">
  Spill av
</PrimaryButton>
```

## Lys bakgrunn

Denne er kun implementert for Strim per nå.

## PrimaryButton
```tsx live=true 
<LightMode style={{
    backgroundColor: "var(--surface-background)",
    padding: "var(--rds-spacing--32)",
    borderRadius: "var(--rds-roundness--default)"
}}>
<PrimaryButton postfix="" isLoading={false} disabled={false}>
  Hello world
</PrimaryButton>
</LightMode>
```
## SecondaryButton

### With 'lightBackground' attribute

## PrimaryButton
