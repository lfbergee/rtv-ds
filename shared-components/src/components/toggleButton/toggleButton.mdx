# ToggleButton

ToggleButton er en fancy checkbox. Den kan brukes til feks å markere og endre om et element er i en gitt status. 
Hvis du har mer enn en toggle i samme view anbefales det å spesifikt sette id på komponenten.

```tsx live=true
<ToggleButton>Direkte</ToggleButton>
```

Hvis du ønsker å bruke komponenten ukontrollert, men ha en start state, bruk defaultChecked-propen.

```tsx render=true
<ToggleButton
  id="toggle-default"
  defaultChecked
>
  Direkte
</ToggleButton>
```

Det er to måter å sette ikon, du kan enten la en toggleIcon, som tar inn `checked` som prop og render riktig basert på det. 
Denne måten er å foretrekke hvis man kan animere mellom to states i samme ikon.

```tsx render=true
<ToggleButton
  id="togge-icon"
  icons={{ toggleIcon: HeartToggle }}
>
  Min liste
</ToggleButton>
```

Eller du kan gi et checked-ikon og et unchecked.

```tsx render=true
<ToggleButton
  id="direct-icon"
  showLabel={false}
  icons={{ checkedIcon: Heart, uncheckedIcon: HeartUnfilled }}
>
  Favoritt
</ToggleButton>
```

## Som uncontrolled component

```tsx
const MyComponent = () => {
  const ref = useRef<HTMLInputElement>(null);

  <ToggleButton ref={ref} defaultValue={false}>
    Favoritt
  </ToggleButton>;
};
```

## Som controlled component

```tsx
const MyComponent = () => {
  const [isChecked, setIsChecked] = useState(false);
  const handleChange = (e) => setIsChecked(e.target.value);

  <ToggleButton value={isChecked} onChange={handleChange}>
    Favoritt
  </ToggleButton>;
};
```
