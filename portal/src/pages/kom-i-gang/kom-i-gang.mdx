# Kom i gang

1. `$` `npm install @rikstv/shared-components`
1. Pakk inn appen din i ønsket merkevare, `<StrimProvider> <App /> <StrimProvider>` eller `<RiksTVProvider> <App /> <RiksTVProvider>`
1. Importer rot stilen `import "@rikstv/shared-components/dist/core/strim.css"` eller `import "@rikstv/shared-components/dist/core/rikstv.css"`
1. Importer stilen til komponentene du trenger feks `import "@rikstv/shared-components/dist/button/Button.css"`
1. Legg til fontene du trenger, kan hentes fra portalen

```tsx
import {
    StrimProvider,
    PrimaryButton,
    DisplayTitle,
} from '@rikstv/shared-components";

import '@rikstv/shared-components/dist/core/Strim.css";
import '@rikstv/shared-components/dist/button/Button.css";
import '@rikstv/shared-components/dist/typography/Typography.css";

const Root = () => (
    <StrimProvider>
        <main>
            <DisplayTitle>
                MyApp
            </DisplayTitle>

            <PrimaryButton>
                Hello world
            </PrimaryButton>
        </main>
    </StrimProvider>
)
```

## Importere stil

Komponentene klarer ikke importere stilen selv. Det betyr at du som tar ibruk komponentene har ansvaret for å sørge for at 
stilen er tilgjengelig for komponenten. Den enkleste varianten er som i eksemplet over og med en `style-loader` til ditt 
byggsystem. Det kan også være et alternativ å laste css-filene inn i `index.html`, gjerne med `preload` så stilen er tilgjengelig 
så fort som mulig.

### Med vite

Vi er veldig glade i Vite, så der har vi gjort litt ekstra for å forenkle prosessen med å ta ibruk komponenter. Legg til i 
`vite.config.ts` 

```ts
import { styleResolver } from '@rikstv/shared-components/dist/tools';

export default defineConfig(() => ({
    plugins: [styleResolver]
}))
```
